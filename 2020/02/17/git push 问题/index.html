git push 问题

git push 之 问题

fatal: TaskCanceledException encountered.

解决：

配置全局量

config --global credential.helper manager

​

fatal: TaskCanceledException encountered.

A task was canceled.

出现这个后，提示输入远程库的账号&密码
，每次push都会出现太麻烦。

解决方法：添加配置文件.gitconfig
在根目录，我的在我的系统盘个人用户下面。

cmd命令行：

git config --global credential.helper store

打开.gitconfig可以看见如下：

[user]
    name = *****
    email = *********@**.com
[credential]
    helper = store

这样设置后，出来第一次输入账号&密码后，以后的提交将不再进行账号&密码的输入。
